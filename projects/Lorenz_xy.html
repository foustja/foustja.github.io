Lorenz Attractor

Juan Carlos Ponce Campuzano provides a nice discussion of the mathematics of the Lorenz Attractor here.

Paul Bourke also has an interesting page on the Lorenz Attractor, including some C code which I borrowed and modified for this P5.js script.

The attractor can be visualized using 3 coupled differential equations:
dx/dt = a(y - x)
dy/dt = x(b - z) - y
dz/dt = xy - cz
In this animation parameter a varies incrementally between 10 and 12.
